✈️ Airline Management System

A desktop-based Airline Management System built using Java Swing and MySQL, designed to manage flight bookings, customers, and seat reservations efficiently with a user-friendly interface.

🚀 Features
👤 Add Customer Details
✈️ View Flight Information
📅 Book Flights
💺 Seat Selection with Real-time Availability
🔒 Prevents Double Booking (Thread-safe logic)
🎫 Generate Boarding Pass
📖 View All Bookings
❌ Cancel Booking (Seat becomes available again)
🕒 Departure & Arrival Time Display
🎨 Modern UI with Toolbar & Background Image
🛠️ Tech Stack
Java (Swing) – GUI
MySQL – Database
JDBC – Database connectivity
NetBeans IDE

🛠️ Tech Stack

- Java (Swing)
- MySQL
- JDBC
- NetBeans IDE

 ▶️ How to Run

1. Open project in NetBeans  
2. Setup MySQL database  
3. Run `Home.java`
